Directions:
1. Open IntelliJ
2. Locate the project to open
3. Open the Main.java file
4. Click the green run button near the top of the screen.

Description of an additional report:
The report I chose determines how many customers are in each Country. In the reports screen, you can select one of the three current Countries.
When a Country is selected, it prints out texts to the right of the selection stating how many customers there are in that Country.
